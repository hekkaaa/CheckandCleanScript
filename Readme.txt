написан на powertshell 5.1.
*** Данный релиз от 17/10/2020.

Общая Информация по работе:
- ПО разместить в C:\Temp\explorefile
- файл для загрузки списка машин C:\Temp\explorefile\Hosts1.csv !!!не удаляем Name из 1 строчки!!! 
- Логи сохраняются по дате запуска в папку explorefile в названии дата и время запуска с локального ПК.
- Копировать код в PS ISE открыть предарительно из под админа.
- Если съехали шрифты то копировать в notepad++ и от туда в ISE. Кодировка windows 1251 исправит русский язык.

****** РАБОТА С ИНТЕРФЕЙСОМ
- В начале скрипта отеделено *ИНТРЕФЕЙС* все переменные которые используются.
- Вписывать значения нужно в кавычках.
- $Programms указывается искомое ПО. Пример: "Putty*"
- $version1 указываем ниже какой версии будет проверенно ПО и удалено и переустановленно. Пример: "7.2.1"
- $Module работа в режиме поиска или поиск + удаление + реинстал. Значения "0" - поиск или "1" - удаление/установка. * Модуль поиска ищет каким образом было установлено ПО MSI или Programs(exe или иной способ)
- $Share - неизменять если не переписываешь скрипт под себя.
- $sourcePath откуда берется дистрибутив.
- $CMDLine1 ключи установки ПО exe формата.
- $CMDLineMSI ключи для установки MSI.
- $UninstalCMDLine ключи на удаление ПО установленное exe файлами.
* MSi удаляются без ключей.

Принцип работы.
***
Вы вписываете ПО по примеру "putty*". Указываете версию ниже которой нужно искать - например "8.8.2". ПО ищет и сообщает в отчет. От указанного модуля зависит что мы получим.
Есть поиск ПО и версии. Есть поиск последующее удаление и установка указанного дистрибутива.

Ключи для установки и удаления exe файлов нужно искать самостоятельно!
Можно воспользоваться https://chocolatey.org/ -> ищем нужное ПО -> если оно есть проваливаемся в него на сайте -> Ищем разде "FILES" -> ищем в списке файл с название uninstal.ps1 или instal.ps1 -> ждем Show -> Смотрим нужные ключи.  


Информация в отчет бывает следующих видов:
- машина не в сети
- ПО есть и оно ниже указанной версии< успешно переустановлено. (переустанавливает ПО на машине)
- ПО есть, оно ниже указанной версии, но запущено и его не переустановить.
- ПО есть и оно соответсвует указанной версии. 
- ПО нет на машине или оно выше версии.
- ПО найдено, оно не соответсвует требуемой версии
- ошибка непредвиденная. (выведется код краша)
- ошибка установки exe/msi файла "BAD - TOTALY FAILED...".
- ошибка удаления программы с указанной машины установленной ранее с помощью msi или exe.
- ошибка формата установочного файла (указан не msi и не exe) "ERROR FORMAT install FILES!".
- ошибка формата providername указанного в удаляемой программе на машине. т.е установка была сделана не с помощью msi или exe файла.
